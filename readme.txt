Για να λειτουργήσει το backend χρειάζεται να τρέξει τοπικά το αρχείο node_client.js, οπότε πρέπει να υπάρχει εγκατεστημένη node.

Συμπεριλάβαμε και ένα postman collection στο οποίο μπορείτε να τρέξετε τα requests και local και σε server που στήσαμε στο Heroku.

Δυστυχώς, δεν προλάβαμε να το κάνουμε configure και στο front-end ώστε να μη χρειάζεται να τρέξει τοπικά η node.js.
